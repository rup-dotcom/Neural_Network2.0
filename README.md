This repository contains solutions to two machine learning problem sets, each addressing real-world problems from different domains.

Problem Set 1: Focuses on medical image classification. A Convolutional Neural Network (CNN) model is developed to detect pneumonia from pediatric chest X-ray images. The objective is to accurately classify images into normal and pneumonia categories, demonstrating the application of deep learning techniques in healthcare diagnostics.

Problem Set 2: Focuses on predictive modeling in the banking sector. A Logistic Regression model is implemented to predict whether a customer will subscribe to a term deposit based on demographic and financial attributes. This illustrates the use of machine learning for customer behavior analysis and decision support in financial services.

Overall, these projects demonstrate the application of both deep learning and traditional machine learning approaches on unstructured and structured data, respectively.
